# Otimizador de PC em Python

## 📌 Descrição do Projeto

Este projeto é um programa desenvolvido em Python com o objetivo de auxiliar na otimização do desempenho do computador. Ele realiza tarefas básicas de análise e limpeza do sistema, ajudando o usuário a manter o PC mais rápido e eficiente.

## ⚙️ Funcionalidades

- Monitoramento do uso de CPU
- Monitoramento do uso de memória RAM
- Verificação de processos ativos
- Limpeza de arquivos temporários
- Geração de relatórios simples sobre o desempenho do sistema

## 🧠 Objetivo

O objetivo principal do projeto é demonstrar como a linguagem Python pode ser utilizada para automação de tarefas de manutenção do sistema, além de ajudar na organização e otimização de recursos do computador.

## 🛠️ Tecnologias Utilizadas

- Python 3.x
- Bibliotecas padrão do sistema operacional (subprocess, sys, etc.)

## 🚀 Como executar

1. Instale o .exe
2. execute

## 📄 Observação

Este projeto é voltado principalmente para fins educativos, servindo como exemplo de automação e gerenciamento de recursos do sistema com Python.
